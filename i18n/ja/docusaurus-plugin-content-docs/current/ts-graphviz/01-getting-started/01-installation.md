# インストール

## Node.js

このパッケージは、パッケージマネージャーを使用してインストールできます。

```bash npm2yarn
npm install ts-graphviz
```

## Deno

以下を指定してパッケージをインポートして使用できます。

```ts
import { toDot } from 'npm:ts-graphviz';
```

:::info
[Deno v1.28 以降は npm をサポートしています。](https://deno.land/manual/node/npm_specifiers).
:::
